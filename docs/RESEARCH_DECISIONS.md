# GRAFT-GS research decision record

This record is normative for the implementation.  Complexity uses `M` image
particles, `V` active charts, `E_OT` transport edges, `E` chart edges, and `F`
faces.  All topology guarantees are conditional on the assumptions recorded in
`MATHEMATICAL_ASSUMPTIONS.md`.

## Architecture decision matrix

| Subsystem | Problem and candidates | Selected formulation | Preserved invariants and rationale | Complexity / memory | Principal failure modes |
|---|---|---|---|---|---|
| Dataset contract | Filename inference, schema-specific loader, manifest-first typed measures, independent modality datasets | Physical-file-verified object manifest with explicit direct/derived/pseudo-label provenance and synchronized variable-view loading | A camera manifest never implies an image exists; continuous alpha is separated from evidence validity; sparse surface voxels are not promoted to volume truth; every camera/image tensor shares one frame index | Manifest audit `O(number of files + array headers + surface vertices)`; per sample `O(KHW+N)` | Unseen schema variation because the supplied train split is empty; incomplete view sets; producer-version drift |
| Corpus discovery | External ID catalog, render-directory enumeration, union of every filename, repeated per-object probing, indexed modality intersection | One-pass per-modality artifact index; candidate union from latent/normalized-mesh primary trees; configurable required intersection plus explicit optional availability | Dataset membership follows physically coexisting producer artifacts rather than an example ID. Optional pseudo-label/surface modalities never fabricate zeros or discard otherwise valid observations. Relative structural maps preserve root relocatability. | `O(F log F)` deterministic indexing for `F` serialized artifacts and `O(NM)` intersection bookkeeping for `N` candidates, `M=9` modalities; avoids the former `O(NF)` repeated walks | Multiple model variants for one modality are intentionally ambiguous unless the corpus is normalized or an explicit selection policy is introduced; a producer may leave syntactically present but semantically corrupt files, which are rejected during audit |
| Cross-model pseudo features | Direct DINO/TRELLIS channel regression, learned MLP adapter, feature concatenation, relational kernel alignment | Confidence-weighted edgewise cosine-kernel alignment after exact surface-cell-to-chart assignment | Learned channel bases of VGGT, DINOv2, and TRELLIS are not identified, so direct coordinate regression is rejected. Pairwise similarity is dimension-independent and gauge invariant when computed from `0e` chart fields; pseudo labels never become geometry or topology truth | Assignment/aggregation `O(N)` and overlap-edge comparison `O(E)`; no dense token attention or cross-feature matrix | Teacher representation bias; charts without two supervised adjacent cells receive no relation; cosine geometry discards feature norm information |
| Evidence lift | Partial OT, KL-unbalanced OT, fused GW, Schrödinger bridge, probabilistic correspondence | Sparse fused geometric KL-UOT with ray-aligned covariance, physical area mass, visibility support, and implicit differentiation | Mass remains a geometric measure; KL relaxation rejects background and permits unseen atlas mass. FGW is reserved as a multiscale regularizer because pairwise intra-measure terms would raise a local solve from `O(E_OT)` to at least quadratic neighborhood coupling. A Schrödinger bridge is unjustified for a static one-step correspondence unless a calibrated stochastic observation process is learned. | `O(E_OT)` state and Sinkhorn work per iteration; chunked support construction avoids `V*M` storage | Poor uncertainty calibration; disconnected support; entropic bias; nearly balanced relaxation slows the implicit solve |
| Supervised gauge and uncertainty | Direct camera replacement, per-view alignment, scene Sim(3), mesh-rasterized depth, quantized surface likelihood | Joint differentiable scene Sim(3) for VGGT geometry plus Gaussian evidence NLL against surface cells with `h^2/12 I` quantization variance | Predicted depth/rays/cameras stay projectively consistent; only global gauge is removed; relative camera error remains; learned SPD covariance cannot collapse onto voxel centers | Camera solve `O(K)` plus one batched 3x3 SVD; exact chunked surface assignment `O(MN)` time and `O(chunk*N)` memory in the reference path | Degenerate camera centers; SVD ambiguity under symmetric rigs; voxel surface misses sub-cell geometry; nearest-sample switches are nonsmooth |
| Transport objective | Entropic OT, debiased Sinkhorn divergence, partial mass cap | KL-UOT plan and its exact sparse primal objective | The positive plan is required for barycentric chart statistics. Debiasing was rejected for the reference path because source-source and target-target support truncations introduce a second, support-dependent approximation while tripling solve cost; it remains an ablation. Partial OT requires a prescribed transported mass that is not identifiable under unknown visibility. | One sparse solve; `O(E_OT)` objective evaluation | Entropic blur when epsilon is too large; assignment brittleness when too small; regularized objective is biased |
| Spatial representation | Fixed sparse voxels, adaptive octree, simplicial complex, neural implicit field, wavelets | Persistent 2:1-balanced octree carrying explicit quadratic charts and a selected simplicial surface complex | Stable Morton identity supports multilevel supervision and checkpoints; charts preserve continuous differential geometry; the simplicial complex carries topology independently of octree occupancy. No isovalue is required. | `O(V+E+F)` state; refinement `O(M log V)` assignment plus local fits | PCA frame ambiguity at isotropic neighborhoods; quadratic charts fail beyond their injectivity radius; incorrect topology proposal cannot be repaired by continuous flow |
| Hidden-surface support | Late prior scores on observed nodes, pseudo-particles in image OT, independent TRELLIS decoding, empirical structured support measure | A Jeffreys-smoothed Beta-Bernoulli measure from multiple TRELLIS structure samples seeds additional canonical atlas cells; observed/prior mass, count, and posterior variance remain separate, image particles remain the only UOT target, and topology occupancy uses a lower-confidence additive surface hazard | The prior can propose genuinely unseen support without becoming an observation, ground-truth label, feature concatenation, or asset decoder. Repeated samples dominate one-off hallucinations; low-retention UOT rows shrink to their atlas charts instead of acquiring arbitrary nearest-image barycenters. | Coordinate union and voting `O(SP log(SP))` for `S` samples with `P` occupied cells; atlas/UOT complexity grows only with admitted support | TRELLIS coordinate-frame mismatch; correlated posterior samples invalidate independent-cell variance aggregation; hallucinated support; an over-strong hazard prior can dominate weak observed geometry |
| Refinement | Isotropic variance split, curvature split, image-plane disagreement, anisotropic metric adaptation, PH-guided refinement | Geometry residual, curvature, occupancy entropy, and sparse plan-conditional image reprojection variance with 2:1 closure; discrete assignment followed by conditionally differentiable chart fitting | Isotropic cells preserve Morton nesting; evidence precision determines the continuous anisotropic metric. Retained scene camera tables make the cross-view statistic exact in the calibrated image model, normalized by projected cell scale. Functional post-split fitting preserves continuous gradients without inventing a gradient for Boolean split indices. | `O(E_OT log E_OT)` grouping plus linear projection/moments and local chart fits; camera storage `O(K)`, not `O(E_OT)` | Calibration/occlusion error can trigger excessive splits; a single-view chart has no cross-view statistic; under-refinement of unobserved thin structures |
| Continuous atlas metric | Nearest chart metric, linear blend, log-Euclidean blend, affine Karcher mean, compact partition of unity | Compact smooth bump partition of unity with a nearest-chart fallback outside support; positive weighted sum of chart SPD metrics | Convex SPD closure gives a simple exact positivity guarantee and SE(3) covariance. Log/Karcher means add iterative matrix functions without improving the node-based flow contract enough to justify their cost. | Reference query `O(QV)` in chunks; no dense state is retained | Nearest fallback is nonsmooth at exterior Voronoi boundaries; badly scaled chart metrics remain badly conditioned |
| Equivariant processing | SE(3) transformer, TFN, e3nn attention, Lie convolution, local-frame gauge network | Connection-aware sparse transport attention over `0e`, `1o`, `2e`, with exact local-frame parallel transport and parity-valid tensor products | Global SE(3) invariance follows from local coordinates; local gauge covariance follows from `P_ji=R_i^T R_j`; multiplicity maps never mix magnetic components. Exact Cartesian symmetric-traceless tensors avoid numerically evaluating Wigner-D matrices for `l=2`. | `O(E*C)` messages, `O(V*C^2)` multiplicity projections; no dense attention | Frame discontinuities if the connection is not synchronized; neighborhoods with no edges; high-order information limited to `l<=2` |
| Equivariant residual stability | Unbounded residual maps, local spectral normalization, exact global Lipschitz analysis, small-step normalized messages | Spectrally parametrized multiplicity maps with an effective configured operator scale, bounded normalized messages, sigmoid gates, and explicit residual scales; global certification still requires a measured composed field bound | Scaling acts only on irrep multiplicities and therefore preserves gauge/SE(3) laws. A real configuration control is preferable to dead `spectral_bound` metadata, while avoiding an unsupported claim that local bounds multiply tightly through dynamic sparse attention. | Spectral power iteration per parametrized map; no added state tensor or representation mixing | One-iteration spectral estimates can be loose during training; tensor/radial paths and heads determine the composed field bound; empirical server calibration remains necessary |
| Topology proposal | Occupancy threshold, PH penalty only, discrete Morse, alpha/witness complexes, learned autoregressive complexes | Structured distribution whose primary cuts are exact lower-star persistence birth/death events ranked by lifetime, with quantile/fixed coverage fallbacks, an evidence-filtered clique surface complex, exact Z2 persistence, incidence/orientability checks, and energy selection | Topology is selected before flow. A support topology can change only at a filtration event, so the bounded budget is spent on critical endpoints rather than arbitrary quantiles first. Betti numbers and boundary operators are exact per candidate; a Z2 face-adjacency solve makes orientation admissibility hard. | One reference persistence reduction plus candidate construction approximately `O(V*d^2)` for degree `d`; bitset persistence near `O(N_cells^2 / word_size)` worst case | Clique triangulation can miss nonlocal faces or fill a true handle; a finite critical-cut family does not enumerate alternate discrete-Morse matchings/connectivities |
| Topology supervision provenance | Raw mesh as truth, manifold repair, differentiable persistence target, teacher pseudo-label, unavailable label | Typed raw/validated/repaired/derived/teacher states with hard masks, nullable targets, and confidence-weighted expected Betti mismatch | Invalid connectivity cannot become `(0,0,0)` or silently activate a loss. Mesh geometry remains usable for rasterization. Persistence requires a declared filtration and stratum labels require atlas correspondence. | Raw audit `O(F log F)` from edge grouping; supervised candidate mismatch `O(C)` for `C` strata | Repair changes topology; teacher bias; a validated source complex can still encode asset artifacts rather than semantic object topology |
| Continuous dynamics | Euclidean rectified flow, manifold diffusion, Schrödinger bridge, Riemannian OT flow, geodesic shooting | Topology-conditioned Riemannian flow matching on positions x SO(3)^V x SPD(3)^V x Euclidean appearance/logits, using explicit geodesics/retractions and exact minibatch assignment only inside identical persistent strata | Rotations remain in SO(3), covariances remain SPD, topology connectivity is fixed. Compatible-stratum Hungarian coupling reduces arbitrary pairing; cross-stratum coupling is rejected because no canonical complex matching is available. | Coupling `O(B_s^3)` per compatible group plus `O(L(E*C+V*C^2))` integration | Cut-locus ambiguity near pi rotations; affine-invariant SPD conditioning; most variable-object batches have singleton stratum groups |
| Hard feasibility | Penalties, log barriers, projected ODE, control-barrier QP, collision-free isotopy solver | Local metric CBF projection, one global speed rescale, explicit primal-margin check, and nonlinear predictor/corrector backtracking; exact KD broad phase under the speed bound | Global positive rescaling preserves all satisfied coupled CBF inequalities for positive state margins. The fixed broad phase contains every pair capable of collision over unit time; every evaluated/accepted integration state is nonlinearly checked. | KD broad phase `O(V log V + J)` expected; reference active dual solve `O(J^2)` time/`O(J^2)` memory | Initial infeasibility; dual nonconvergence; dense collision neighborhoods; shortened predictor makes the safeguarded scheme depart from ideal fixed-step Heun |
| Gaussian readout | Neural decoder, free 3DGS optimization, per-voxel Gaussian, differential surface sampling | Deterministic curvature-quadrature chart sampling; covariance from first form/curvature plus continuous evidence uncertainty; optical-depth opacity; weighted-ridge SH; conservative projected-tile opacity bound | Centers lie exactly on the atlas, covariance is SPD by construction, opacity lies in `(0,1)`, and appearance solves a convex analytical problem. The tile sum is a true upper bound because every retained footprint is at most one. | Sampling/readout `O(G + E_obs*16^2)` plus bbox-tile incidence | Sparse views make higher SH orders prior-dominated; overlap can still oversample; tile bound may be loose |
| Mesh/asset output | Marching cubes, independent mesh decoder, Poisson reconstruction, atlas complex | PLY Gaussians and GLB mesh serialized from the same selected atlas state and topology complex | No implicit threshold; deterministic byte order; mesh/Gaussian geometry cannot diverge through separate decoders | `O(G)` PLY and `O(V+F)` GLB | A wrong selected complex remains wrong in both outputs; GLB only approximates curved chart interiors unless subdivided |
| Distributed training | Data parallel objects, model parallel backbone, sharded nonlinear OT, replicated global OT, asynchronous topology | DDP model replication; same-object mode autograd-all-gathers the complete evidence measure and replicates one global sparse UOT/atlas solve; discrete state is verified/broadcast by persistent keys | A sum of local UOT barycenters is not falsely identified with a global nonlinear UOT solution. Every rank sees the same evidence, support, Morton identities, and selected stratum; DDP averaging cancels the all-gather backward sum | Communication `O(M*(D+3x3))` and redundant global solve per rank in the reference path; ordinary object DDP uses normal gradient buckets | High feature communication and duplicated compute; object-size stragglers; collective deadlock on rejected samples; future sharded solver requires equivalence proof |
| Robust multiview gradients | Arithmetic mean, coordinatewise clipping, geometric median only, full parameter-space SVD, gradient sketches, Gram-space robust subspaces | Phase-F weighted Hilbert geometric median, exact circular-cone projection, exact small-Gram consensus/artifact projectors, then diagonal empirical-Fisher clipping; only view-conditioned losses enter, while global structural gradients remain exact | Parameter grouping is invariant to tensor layout, the cone rejects opposing views, the consensus projector retains jointly supported directions, deterministic color/mask perturbations estimate nuisance directions, and DDP averages only after local purification | `O(K^2 P + K^3)` arithmetic and `O(KP)` retained gradients for `K<=8`, trainable parameter count `P`; no dense `P x K` matrix is allocated | Extra backward passes; Fisher diagonal ignores parameter correlations; artifact perturbations can overlap legitimate appearance learning; consensus fails if most views are wrong |
| Topology-margin adversary | Random perturbation only, simultaneous descent/ascent, persistent adversary, one-step FGSM, multi-step PGD | One-step signed inner maximization of bounded per-block log quantization scales with an RNG-replayed full forward, combined with random camera/depth/confidence/pruning/view perturbations and dimensionless positive certificate slack | The second forward differs only in bounded quantization scale, not stochastic rounding or sampled geometry noise. Hardening loss compares like-with-like normalized margins and retains the exact barrier as zero slack. Multi-step PGD was rejected for the reference path until one-step equivalence and memory are measured. | Two full Phase-F forwards plus scalar scale gradients; adversary state is `O(number of quantized blocks)` | FGSM can under-estimate curved inner objectives; discrete quantization decisions are nonsmooth; random camera perturbations are not a solved worst-case pose attack |
| Quantization topology certificate | Caller scalar margin, Euclidean constraint slack, sampled boundary distance, exact metric constraint gradients | Exact piecewise `min_j h_j/(sqrt(dh_j G^-1 dh_j)+eps)` over the retained position-constraint family, combined with the analytical normalized-score error bound and an explicit externally measured downstream Lipschitz upper bound | The topology distance has consistent displacement units across area, orientation, and squared-separation constraints. No margin or end-to-end Lipschitz constant is fabricated from training losses or local spectral normalization. | Reference gradient loop is linear in constraint count and avoids `O(JV)` retained Jacobian memory; intended for evaluation/certification, not every training microstep | Closest-feature/broad-phase switching is conditional; a loose empirical Lipschitz upper bound makes the certificate vacuous; missing A800 measurement disables rather than passes certification |
| Deep distillation | Final-state only, packed-feature MSE, layerwise gauge loss, dense Jacobian, Hutchinson/JVP matching | World-tensor comparison of all captured `0e/1o/2e` encoder stages plus one deterministic normalized product-manifold JVP; teacher tangents are parallel-transported to the student SO(3), affine-SPD, and irrep gauges | No coordinate-dependent packed-vector MSE is used. Affine transport preserves the SPD metric exactly, and the JVP tests local dynamics without allocating a quadratic Jacobian. One deterministic probe was selected over random Hutchinson probes for exact resume and lower variance across teacher/student. | Activation memory `O(LVC)` in Phase E only; two directional vector-field derivative evaluations instead of a dense state Jacobian | One probe can miss an anisotropic Jacobian error; differing atlas rows make activation loss inadmissible; JVP double-backward raises Phase-E memory |
| VGGT geometric self-supervision | Invented track/normal heads, optical flow teacher, nearest image features, depth-camera cycles, depth finite-difference normals | Deterministic sparse source-depth unprojection, target projection/bilinear depth sampling, world/source cycle closure, and confidence/visibility weighting; neighboring depth rays define a detached world-normal pseudo-target | Uses only released VGGT outputs and audited camera conventions. Track loss remains differentiable to depth/cameras; detached normals prevent renderer/VGGT co-adaptation. No absent modality is silently zero-filled. | Track reference `O(B K^2 HW/s^2)` at stride `s=16`; normal field `O(BKHW)` | Occlusions and repeated surfaces can defeat depth visibility weights; finite differences fail at depth discontinuities; predicted cameras/depth can share a biased self-consistent solution |
| Offline teacher | Independent Gaussian optimization, mesh repair, unconstrained bundle adjustment, topology-fixed atlas bundle adjustment | Robust Cauchy rendering fit over bounded camera SE(3)/intrinsics and a product-manifold tangent to the selected atlas state; hard feasibility backtracking after every optimizer update; common analytical Gaussian/mesh readout and confidence/provenance bundle | Topology cannot change during continuous refinement, all emitted assets share the atlas, and low-consensus teachers are masked rather than called truth. Direct raw-mesh topology never enters the teacher state. | `O(T)` full readout/renders for `T` offline iterations; no student-inference cost | Renderer local minima; topology initialization error is frozen; photometric ambiguity; reference backtracking can shorten large parameter updates; checkpoint execution is external |
| Perceptual supervision | Pixel loss only, fixed gradients, downloaded VGG, pinned VGG, LPIPS | Optional local SHA-256-pinned frozen VGG16 feature pyramid with foreground-mask resampling; deterministic color/gradient pyramid when absent | No runtime network or mutable model hub state enters the objective. Checkpoint identity is exact-resume policy. LPIPS was rejected as a claim because no trained LPIPS calibration/checkpoint is supplied. | Four frozen VGG stages during render phases; activation memory scales with `B K H W` | External checkpoint quality/provenance; VGG domain bias; added Phase-F per-view backward memory; fixed fallback is weaker |
| TRELLIS topology prior | Fuse prior into evidence, candidate-index network, independent-cell Bernoulli, decoded mesh prior | TRELLIS/observed hazard union proposes candidate support; UOT-only occupancy defines evidence/persistence; a Jeffreys-smoothed per-active-cell Bernoulli likelihood scores each candidate under a separate weight | Hidden prior never becomes an observation. Zero TRELLIS votes imply finite absence evidence rather than certainty, and a strong observed likelihood can override the shape prior. A candidate-index network was rejected because candidate ordering is data-dependent and has no pretrained calibration. | `O(CV)` candidate NLL for `C<=12`; no additional decoder | Independent-cell factorization ignores structured correlations; prior resolution/atlas aggregation mismatch; correlated TRELLIS samples overstate effective evidence |
| Precision | FP4 everywhere, BF16 geometry, mixed native precision | BF16/FP16 backbones and attention projections; FP32 geometry, OT potentials, manifold maps, barriers, solves, and export; optional FP64 diagnostics | Topological and SPD margins are not quantized during the native baseline. Low-bit execution is introduced only after equivalence tests and margin certificates. | Higher state memory is acceptable on 80 GB A800s | FP32 may still be insufficient for nearly singular charts; use FP64 diagnostic path |
| Zero-residual geometric norms | Raw Euclidean norm, squared cycle, epsilon-shifted norm, debiased Charbonnier | `sqrt(||r||^2+eps^2)-eps` for world and pixel cycle residuals | It is exactly zero for a closed cycle, locally smooth with finite derivative, and asymptotically Euclidean; raw `sqrt(sum(r^2))` has an undefined zero derivative that produced NaNs in PyTorch 2.4. Squared error changes physical scaling and outlier influence. | Constant work and no added state | `eps` defines a tiny quadratic basin; calibration remains conditional on depth visibility |
| Server environment identity | Import smoke test, loose minimum versions, exact lock audit | Fail-fast comparison of installed distribution metadata with every unconditional `==` pin plus `pip check` | Numerical equivalence, compiled ABI behavior, and the supplied validation result are only interpretable against the declared environment. The audit is read-only and records the requirements digest. | Linear in installed and required distribution count; executed before model import | A pin set does not prove CUDA driver or hardware identity; those remain runtime reports |

## Selected end-to-end factorization

```text
VGGT multiscale geometry taps + heads
  -> calibrated evidence measure
  -> sparse implicit KL-UOT
  -> persistent metric-adaptive chart atlas
  -> connection-aware gauge transport attention
  -> structured topology candidate distribution
  -> selected fixed simplicial stratum
  -> barrier-projected product-manifold flow
  -> analytical surface-attached Gaussians + shared atlas mesh
  -> differentiable rendering + deterministic PLY/GLB
```

The defining architectural boundary is between discrete topology inference and
continuous manifold refinement.  Neither a topology loss nor a continuous flow
is allowed to silently cross that boundary.
